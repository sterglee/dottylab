Matlab .mat file compatibility functions

ScalaLab can easily read Matlab .mat files.

Use:

a. load(filename:String)

to read the contents of a .mat file

Suppose that we have the Matlab workspace created with the Matlab script:

Matr = rand(8, 9)

OnesM = ones(9, 15)

randV = rand(90)



We save that workspace with the Matlab save command in c:\testMat directory. Matlab creates the file matlab.mat in that directory.

Then we load the file from ScalaLab as:

load(“c:\\testMat\\
