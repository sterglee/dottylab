The ScalaLab Editor

The ScalaLab Editor aims to serve as a specialized editing tool integrated with the rest of the environment.

The editor extends the JPanel Swing component and thus it has the flexibility of placed within other components with the utilization of Swing component layout organizers.

User actions of the editor are modeled by classes implementing the Action interface. This interface requires to provide code for the actionPerformed routine.

The editor explores the DefaultEditorKit standard Swing class for providing standard operations like cutting, copying, pasting text and undoing/redoing.
