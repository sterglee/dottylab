Useful   Input   Commands 


The following routines are very useful for performing input:
// gets a double value with a dialog box, specifying an optional default parameter
 def getDouble(messageOfDialogBox: String, [defaultValue: Double]): Double 





// gets an int  value with a dialog box, specifying an optional default parameter
 
def   getInt(messageOfDialogBox: String, [defaultValue: Int]): Int  

 
 //gets a String value with a dialog box, specifying an optional default parameter
 
def getString(messageOfDialogBox: String, [defaultValue: String ]): String
 



// gets a File with a File Dialog, starting at directory initialDirectory, specifying optionally a message to display to the
 
def getFile(initialDirectory: String , [messageToUser: String ]): String  user
