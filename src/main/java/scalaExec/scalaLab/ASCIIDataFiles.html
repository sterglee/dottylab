Reading/Writing ASCII Data Files


You can read/write double arrays to ASCII data files with the following routines:




def saveAscii(d: Array[Double], fileName: String): Unit  

Example: 

var d = vrand(10).getv  // a random Array[Double]
saveAscii(d, "dv.txt")   // save to file dv.txt


  
def saveAscii(dd: Array[Array[Double]], fileName: String, sepChar: Char= ' '): Unit  

Example:

var dd = Rand(10, 20)   // a random Array[Array[Double]]
saveAscii(dd, "ddv.txt")    
    
def readAscii( fileName: String): Array[Array[Double]]     
      
Example:

var rd = readAscii("dv.txt")   // read back the saved 1-D array
var rdd = readAscii("ddv.txt")   // read back the saved 2-D array
